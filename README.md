# jwt-auth
This project is Jwt Authentication and Authorization.

**Features**
1. Secure JWT-based authentication and authorization  
2. Access & refresh token generation  
3. Token validation middleware  
4. Clean and modular architecture  
5. DTO-based request and response models (`UserLoginDto`, `TokenResponseDto`)  
6. Async operations for better scalability  
7. Custom error handling for invalid login attempts  


**Technologies Used**
1. ASP.NET Core / .NET 8
2. Entity Framework Core (optional for persistence)
3. JWT (System.IdentityModel.Tokens.Jwt)
4. Dependency Injection
5. Newtonsoft.Json / System.Text.Json
