# jwt-auth
This project is Jwt Authentication and Authorization.

**Features**

Secure JWT-based authentication and authorization

Access & refresh token generation

Token validation middleware

Clean and modular architecture

DTO-based request and response models (UserLoginDto, TokenResponseDto)

Async operations for better scalability

Custom error handling for invalid login attempts

**Technologies Used**

ASP.NET Core / .NET 8

C#

Entity Framework Core (optional for persistence)

JWT (System.IdentityModel.Tokens.Jwt)

Dependency Injection

Newtonsoft.Json / System.Text.Json
